## 🎨 自定义你的游戏

### 修改游戏配置
打开 `assets/js/game.js`，找到 `CONFIG` 对象：

```javascript
const CONFIG = {
    gravity: 0.6,              // 重力（越大下落越快）
    powerSpeed: 2,             // 蓄力速度
    maxPower: 100,             // 最大蓄力值
    ballRadius: 15,            // 小球大小
    platformWidth: 80,         // 平台宽度
    platformHeight: 20,        // 平台高度
    minPlatformGap: 100,       // 最小平台间距
    maxPlatformGap: 200,       // 最大平台间距
    colors: [...]              // 颜色主题
};
```

### 添加新皮肤
在 `game.js` 中找到 `this.skins` 对象，添加新皮肤：

```javascript
this.skins = {
    // ... 已有皮肤
    newskin: {
        name: '新皮肤',
        icon: '🎮',
        price: 300,
        color: '#ff0000'
    }
};
```

## 🛠️ 常见问题

### Q1: 游戏无法打开？
A: 确保用浏览器打开 `index.html` 文件

### Q2: 如何修改游戏难度？
A: 在 `game.js` 中调整 `CONFIG.gravity` 和 `platformGap` 参数

### Q3: 广告不显示？
A: 需要先注册广告平台账号并集成SDK（参考 `analytics.js`）

### Q4: 如何打包成手机APP？
A: 可以使用 Cordova、Capacitor 或 Electron

### Q5: 可以添加多人对战吗？
A: 可以！需要添加WebSocket服务器和实时同步功能
